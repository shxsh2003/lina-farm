# LINA Poultry Farm DSS

## Project info

LINA Poultry Farm Decision Support System (DSS) built with Vite, React, TypeScript, shadcn-ui, and Tailwind CSS.

## Getting started

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```
